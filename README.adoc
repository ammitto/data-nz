== Purpose

This repository contains sanctioned individuals, entities, and vessels from the
New Zealand Russia Sanctions Register, maintained by the Ministry of Foreign
Affairs and Trade (MFAT).

== Source

* **Authority**: New Zealand Ministry of Foreign Affairs and Trade (MFAT)
* **URL**: https://www.mfat.govt.nz/en/trade-and-country-services/sanctions/
* **XLSX Download**: https://www.mfat.govt.nz/assets/Countries-and-Regions/Europe/Ukraine/Russia-Sanctions-Register.xlsx
* **Format**: XLSX (multiple sheets)
* **Update Frequency**: As updated by MFAT
* **Country Code**: NZ

== Legal Basis

These sanctions are implemented under:

* Russia Sanctions Act 2022
* Russia Sanctions Regulations 2022

Note: New Zealand also implements UN sanctions, but those are covered by the
https://github.com/ammitto/data-un[data-un] repository.

== Data Contents

The register contains three types of designated persons:

* **Individuals**: Natural persons subject to sanctions
* **Entities**: Organizations and companies subject to sanctions
* **Ships**: Vessels subject to ship bans

== Data Format

Individual:

[source,yaml]
----
type: Individual
unique_identifier: IND-1
first_name: Vladimir
middle_names: Vladimirovich
last_name: Putin
dob: '1952-10-07'
title: President of the Russian Federation
sanction_status: Sanctioned
date_of_sanction: '2022-03-18'
travel_ban: 'Yes'
asset_freeze: 'Yes'
aircraft_ban: 'Yes'
ship_ban: 'Yes'
----

Ship:

[source,yaml]
----
type: Ship
unique_identifier: SHP-1
imo_number: '9288693'
name: Andaman Skies
date_of_sanction: '2025-06-19'
ship_ban: 'Yes'
----

== Knowledge Graph Integration

This repository's processed YAML data feeds into the Ammitto knowledge graph.

When the harmonization process runs:

[source,shell]
----
# Auto-detect all data-* repos and harmonize
ammitto harmonize --scan --sources-dir /path/to/repos --output-dir ./api/v1
----

The nz data is transformed into:

* Individual JSON-LD nodes in `api/v1/node/entity/nz/{id}.jsonld`
* Sanction entries in `api/v1/node/entry/nz/{id}.jsonld`
* Lightweight search index entries
* Facet counts for filtering
